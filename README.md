# GuestFlow 🏨

GuestFlow is an AI-powered Hotel Management System that helps hotels manage rooms, guests, and bookings efficiently. It is designed for hotel managers and receptionists, with intelligent room recommendations based on guest preferences, room availability, and pricing.

---
## ✨ Features

### Current
- Responsive modern UI
- Home, Rooms, Dashboard & Login pages
- Room Management interface
- Reusable React components
- Mobile-friendly design

### Planned
- Guest Check-in & Check-out
- Guest History
- AI Room Recommendation
- Authentication & Role-based Access
- Booking Management
- Dashboard Analytics

---

## 🚀 Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
cd frontend
npm install
```

Run the development server

```bash
npm run dev
```
